Projects

Below are the four projects that I completed during my time at General Assembly. I have included the Github links for more details of each project, including the technologies used.

Can I Kick It?

Javascript Game
View Website View on Github

This was my first project on WDI, to understand the basic fundamentals of JavaScript. Klick it is a game where coloured squares appear at random and you gain points for clicking on correct squares, and lose points for clicking on incorrect squares.

Snowglobe

Node/Express Google Maps App
View Website View on Github

An authenticated app using the Google Maps, Google GPX Express, OpenSkiMaps and OpenWeatherMap API’s. The app allows users to look at different ski resorts, see the current weather and weather forecast and look up flight details. Written in jQuery, this was my first experience working with an external API.

ClubMate

MEAN Stack App
View Website View on Github

A team project, we built a RESTful app where users add events or activities that other users can search and request to join. The project involved heavy pair programming, regular team stand-ups and working collaboratively using Git. A MEAN stack authenticated app using AngularJS as a front-end framework.

Musync

Rails/Angular
View Website View on Github

My final project on WDI using a Ruby of Rails backend and AngularJS front-end framework. Using the Spotify API and Web Sockets, users have visibility to music ‘rooms’ they are authorised to enter, and the track playing syncs for all users in the room.
